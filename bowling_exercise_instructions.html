Computer Graphics - Exercise 5

Bowling Alley with WebGL

Spring Semester 2026

Overview

In this exercise, you will implement a 3D bowling alley scene using WebGL and Three.js. You will create a realistic bowling lane with proper markings, a set of ten pins in regulation formation, and a bowling ball with finger holes. This exercise focuses on building the static infrastructure only.

MANDATORY REQUIREMENTS - HW05 INFRASTRUCTURE

Your implementation MUST include ALL of the following infrastructure elements:

  1. A properly detailed bowling lane with at least the following markings:
    • Foul line (white or red line across the lane width)
    • Approach dots (two rows of dots on the approach area)
    • Lane arrows (targeting arrows embedded in the lane)
    • Gutters on both sides
    • Approach area (slightly different color/shade from the lane)
  2. Ten bowling pins in triangular formation, each with:
    • Proper pin shape (wider body, narrower neck, rounded top)
    • White color with red stripe markings
    • Correct triangular arrangement (1-2-3-4 row pattern)
    • Pin deck area (distinct surface behind the pins)
    • Proper spacing (~1 unit / 12 inches center-to-center)
  3. A static bowling ball with:
    • Three finger holes (two adjacent, one offset)
    • Glossy/shiny material appearance
    • Positioned on the approach area (static, no rolling)
    • Proper size relative to lane and pins
  4. Camera and lighting infrastructure:
    • Appropriate lighting with shadows
    • Interactive camera controls (orbit)
    • Appropriate initial camera positioning
  5. Basic UI framework:
    • HTML container elements for future bowling scorecard display
    • HTML container elements for future controls display
    • Basic CSS styling for UI elements

Note: Interactive controls, physics-based ball rolling, pin collision mechanics, and the scoring system will be implemented in the next exercise (HW06).

SPECIFIC IMPLEMENTATION DETAILS

Your bowling alley must meet these specific requirements:

1. Lane Dimensions and Appearance

  • Lane surface should be a light maple/wood color with a glossy finish
  • Lane proportions approximately 17:1 (length:width) — about 60 units long, 3.5 units wide
  • Foul line must be clearly visible (white or red, thin line across full width)
  • Approach area (~15 units long) should be a subtly different color/shade from the lane
  • Gutters must be slightly lower than the lane surface and run the full lane length
  • Lane arrows positioned approximately 15 units from the foul line

2. Bowling Pins

  • Each pin must be approximately 1.25 units tall
  • Pin body should be wider at the base, narrow at the neck, with a rounded top (use multiple geometries or THREE.LatheGeometry)
  • White with at least one red stripe/band around the neck area
  • Triangular formation: 1 pin in front row, 2 in second, 3 in third, 4 in back row
  • Pin spacing ~1.0 unit center-to-center
  • Head pin positioned approximately 57 units from the foul line

3. Static Bowling Ball

  • Ball radius approximately 0.45 units (diameter ~0.9 units)
  • Three finger holes visible on the surface (two close together, one offset)
  • Finger holes can be small dark cylinders embedded slightly into the sphere surface
  • Glossy material with high shininess
  • Positioned on the approach area, centered on the lane
  • NO physics or movement

4. UI Framework Preparation

  • Create HTML containers for future bowling scorecard display (10-frame format)
  • Create HTML containers for future controls instructions
  • Implement basic CSS styling for UI elements
  • Position UI elements appropriately on screen
  • Ensure camera controls are toggleable with 'O' key

Pin Position Reference

The 10 pins are arranged in a standard triangular formation (1-2-3-4 rows). Using the coordinate system where the foul line is at Z=0 and pins are at negative Z values:

Pin 1 (head pin): X= 0.0, Z= -57.000 Pin 2: X= -0.5, Z= -57.866 Pin 3: X= 0.5, Z= -57.866 Pin 4: X= -1.0, Z= -58.732 Pin 5: X= 0.0, Z= -58.732 Pin 6: X= 1.0, Z= -58.732 Pin 7: X= -1.5, Z= -59.598 Pin 8: X= -0.5, Z= -59.598 Pin 9: X= 0.5, Z= -59.598 Pin 10: X= 1.5, Z= -59.598

Pin numbering follows the standard bowling convention:

7 8 9 10 (back row - 4 pins) 4 5 6 (third row - 3 pins) 2 3 (second row - 2 pins) 1 (head pin - closest to bowler)
